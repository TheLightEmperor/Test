Apa Itu Pemrograman?

Pemrograman adalah proses penyusunan, pemeliharaan serta modifikasi untuk dapat memberikan perintah kepada mesin atau komputer.

Fungsi Pemrograman!

Fungsi pemrograman dapat memberikan instruksi pada komputer sehingga user mampu memerintahkan mesin secara spesifik, apa yang harus dilakukan.

Contoh Bahasa Pemrograman Paling Umum:

  • Javascript
  • PHP
  • Pemrograman C++
  • Visual Basic
  • HTML
  • Java
